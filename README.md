# ¿Quién paga la cuenta?

Visualización interactiva de datos del IRPF en España, basada en datos oficiales de la Agencia Tributaria (Ejercicio 2023).

## Datos clave

- **54,5%** de los declarantes solo paga el **6,8%** del IRPF
- **4 de cada 10** declarantes pagan exactamente **0 euros**
- **14.738 personas** pagan más IRPF que 13 millones de personas juntas
- El **5,6% más rico** aporta el **42,3%** de toda la recaudación

## Fuentes

- [Agencia Tributaria](https://sede.agenciatributaria.gob.es/) — Estadística de los declarantes del IRPF, Ejercicio 2023 (Cuota líquida incrementada total, partida 587)
- [RTVE](https://www.rtve.es/) — Campaña de la Renta 2023–2024
- [Business Insider España](https://www.businessinsider.es/)

## Tecnología

HTML + CSS + JavaScript (D3.js). Sin dependencias de build. Un solo archivo `index.html`.
